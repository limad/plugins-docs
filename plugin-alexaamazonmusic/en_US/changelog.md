# Changelog

## Version 0.0.1 (2026-04-23)

First release of the Alexa Amazon Music plugin.

### New features

- Amazon Music playback control on Alexa devices (play, pause, stop, next, previous, fast forward, rewind)
- Volume control via slider (0–100)
- Amazon Music playlist playback from an automatically scanned list
- Play an Amazon Music track by name or ID
- Play a TuneIn radio station by name
- Send arbitrary text commands to Alexa
- Transfer playback session to another Alexa device (Cast)
- Seek within the current track (seekPlayer)
- Repeat mode and shuffle mode management
- Equalizer control: bass, mid, treble (-6 to +6)
- Info commands: volume, repeat/shuffle state, equalizer values, current track duration
- WHA (Whole Home Audio) group support for multi-room audio
- Automatic device import via the alexaapiv2 plugin scan
- Automatic playlist list refresh (~every 17 minutes) with a manual refresh button
